

# Temperature Converter Website

A simple and responsive Temperature Converter Web App built using **HTML & CSS**.  
This app allows users to convert temperatures between **Celsius, Fahrenheit, and Kelvin**.

---

## Features

- Convert temperatures between Celsius, Fahrenheit, and Kelvin  
- User-friendly interface  
- Clean and responsive layout  
- Instant results upon input  

---

## Technologies Used

- HTML5  
- CSS3  
- VS Code (for coding and file management)
